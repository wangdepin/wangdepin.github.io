基因组数据处理-抽取intron序列从gtf文件和基因组文件

我想从gtf文件中抽取每个基因的intron序列。

if you have genes gtf file, convert it to bed format by gtf2bed utility from bedtools then make a genes file in bed (chrom gene_start gene_end from gtf)

substractBed -a genome.bed -b converted_gtf_to_bed.bed > introns.bed

bedtools getfasta [OPTIONS] -fi <genome.fasta> -bed <introns.bed> -fo <introns.fasta>

参考资料：
https://github.com/irusri/Extract-intron-from-gff3
